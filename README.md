## Features

- [x] Vite 打包/构建
- [x] 全局状态管理 gluang
- [x] 支持路由
- [x] 支持 JSX/TSX 语法
- [x] 支持Less/Scss
- [x] 支持 SEO
- [x] 打包产物体积极小，性能高
- [x] 同时支持 TypeScript/JavaScript
- [x] 支持 Eslint/Tslint，代码保存自动格式化等

## Develop & Installation

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## 适合谁？

对性能有极致要求的 Web 开发者～

无框架，没有Vue、没有React，只要原生！！

ps：虽然无框架，但拥有良好开发体验！
